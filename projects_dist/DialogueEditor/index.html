Dialogue editor (0.6)

Condition ⓘ
The condition has to be verified to display the option text. Empty condition means the dialogue has no specific requirements.


Option text (short) ⓘ
The option which will get displayed to the player if the condition is verified, all the verified options will be displayed


Dialogue text ⓘ
The text which will be outputted as the player chooses this exact option


Trigger ⓘ
What happens after this dialogue route gets chosen


Page ⓘ
Click "+" to add a new page, right click the page number to delete it (following pages will refill the emptied space)

Output ⓘ
